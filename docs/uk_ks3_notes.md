\## Key Stage 3 Objectives Relevant to HTML/Web Development



\### 1. Creative Projects (Mini Website Development)

> "undertake creative projects that involve selecting, using, and combining multiple applications, preferably across a range of devices, to achieve challenging goals, including collecting and analysing data and meeting the needs of known users"



\*\*Relevance:\*\* Building a complete HTML website is a "creative project" that meets this objective.



\### 2. Digital Artefacts (Semantic HTML \& Design)

> "create, re-use, revise and re-purpose digital artefacts for a given audience, with attention to trustworthiness, design and usability"



\*\*Relevance:\*\* Web pages are "digital artefacts". Learners must consider design and usability when building HTML pages.



\### 3. Programming Languages \& Data Structures (HTML Tags, Lists, Tables)

> "use two or more programming languages, at least one of which is textual, to solve a variety of computational problems; make appropriate use of data structures \[for example, lists, tables or arrays]"



\*\*Relevance:\*\* HTML is a textual "language". Lists and tables are data structures used in HTML.



\### 4. Data Representation (Images and Multimedia)

> "understand how data of various types (including text, sounds and pictures) can be represented and manipulated digitally, in the form of binary digits"



\*\*Relevance:\*\* This explains how images, sounds, and text are stored and displayed in web pages (multimedia).



\### 5. Computational Abstractions (Website Design)

> "design, use and evaluate computational abstractions that model the state and behaviour of real-world problems and physical systems"



\*\*Relevance:\*\* A website is an "abstraction" that models a real-world need or problem.

\## Summary

The UK KS3 curriculum does not explicitly mention "HTML" by name. However, \*\*5 objectives directly relate\*\* to web development skills:

1\. Creative projects → Website building

2\. Digital artefacts → Web pages with design and usability

3\. Programming languages → HTML (textual language)

4\. Data structures → Lists and tables in HTML

5\. Data representation → Images and multimedia
