

\# Raspberry Pi Foundation - Web Development Projects Notes



\## Source

\- Path: Introduction to Web Development

\- URL: https://projects.raspberrypi.org/en/pathways/web-intro

\- Accessed:21-07-2026



\## Project Overview

\- The path contains six projects\[reference:9].

\- Structured into three Explore projects, two Design projects, and a final Invent project\[reference:10].

\- Introduces semantic HTML for accessibility\[reference:11].

\- Covers responsive design and colour contrast\[reference:12].



\## Project Details





|<br />Project|Type|Key HTML Concepts|Key CSS Concepts|Difficulty|
|-|-|-|-|-|
|1. Anime Expression|Explore|<header>, <main>, <footer>, <section>, <h1>/<h2>, <p>, <img>, <strong>, comments|Classes (primary, secondary, tertiary), borders, text-align, rounded, dashed-border, colour variables, hex colours, responsive wrap, colour palettes|Beginner|
|2. Top 5 Emojis|Explore|<ol>/<ul>, <li>, <blockquote>, <cite>, <a> (hyperlinks), <span>, emojis in HTML|@keyframes (animations), rotate-center, bounce, rollmeleft, spinme, opacity/transparency, wide/narrow classes, responsive design|Beginner|
|3. Flip Your Card|Explore|<div> grouping, <section>, <title>|transform/perspective (3D flips), transition, gradient backgrounds, ycenter (vertical centring), rounded, shadow, @import Google Fonts|Beginner|
|4. Mood Board|Design|Multiple <section> layouts, <div> containers, placeholder text (lorem ipsum), <blockquote>|Colour palettes (20 options), custom CSS classes, gradients, rounded corners, shadows, xcenter/ycenter, transparency, responsive wrap|Intermediate|
|5. Sell Me Something|Design|Landing page structure, <header>, <main>, <footer>, <div> containers, <h1>/<h2> headings, links, lists|Animation (spinme, bounce), flip cards, gradients, custom colour palettes, background images with transparency, shadows, borders|Intermediate|
|6. Build a Webpage|Invent|Complete webpage structure, <title>, <header>, <main>, <footer>, <section> layouts, lists (ordered/unordered), links, quotes, comments, emojis|Full CSS toolkit: colour palettes, gradients, animations, flip cards, responsive design (wrap), custom CSS classes, accessibility (alt text, colour contrast), Google Fonts|Advanced|
